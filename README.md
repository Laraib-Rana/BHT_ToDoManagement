# 🛒 BHT SmartShoppingStore: Interactive E-Commerce Interface

A dynamic front-end shopping store application simulating a modern web-retail environment with real-time cart functionality.

## ✨ Features
* **Dynamic Product Catalog:** Displays a cleanly organized grid of catalog items with interactive details.
* **Live Shopping Cart Engine:** Add items, modify item quantities inline, and watch your total costs recalculate instantly.
* **State Persistence:** Preserves your active shopping cart selections across browser page reloads using web storage.

## 🛠️ Tech Stack
* HTML5 & CSS3
* Bootstrap 5 Component Layouts
* Vanilla JavaScript (DOM Manipulation & State Management)
