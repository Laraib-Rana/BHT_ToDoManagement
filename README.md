# 📝 BHT ToDoManagement: Advanced Task Manager

A robust daily productivity and task management application equipped with full lifecycle controls, real-time filtering, and persistent data tracking. Built as part of the front-end web development internship program.

## ✨ Features
* **Full Task Lifecycle (CRUD):** Seamlessly append, inline-edit, toggle completion status, or delete task rows via a clean, responsive interface.
* **Search & Filter Subsystem:** Dynamically filter tasks by status (All, Active, Completed) or search text instantly.
* **Data Persistence Engine:** Integrated local storage hooks ensure your curated task list survives active page reloads or tab closures.
* **UI Theme Toggle:** Features a smooth light/dark mode presentation switcher for enhanced user accessibility.

## 🛠️ Tech Stack
* HTML5 & Modern CSS3
* Bootstrap 5 Utility Toolkit
* Vanilla JavaScript (ES6+ Engine & LocalStorage Web Hooks)
